# 🛠️ 技能库 — Skills

> 技术栈、方法论、操作手册、踩坑记录

---

## 什么是"技能"？

- **技术能力** — 编程语言、框架、工具
- **方法论** — 调试方法、设计原则、学习策略
- **操作手册** — 环境搭建、部署流程、命令速查
- **踩坑记录** — 遇到的坑和解决方案

---

## 📂 分类建议

```
6-技能库/
├── 编程语言/     # Python、JavaScript、Go...
├── 工具链/       # Git、Docker、Vim...
├── 系统运维/     # Linux、网络、数据库...
├── 架构设计/     # 设计模式、系统架构...
├── 软技能/       # 沟通、写作、时间管理...
└── 踩坑录/       # 每个坑一篇，记录症状+根因+解法
```

## 🕳️ 踩坑录

每个坑一篇独立笔记，格式：

```markdown
---
tags: [踩坑, 技术标签]
date: YYYY-MM-DD
---

# 问题一句话描述

## 🩺 症状
- 发生了什么

## 🔍 根因
- 为什么会发生

## 💊 解法
1. 步骤一
2. 步骤二

## 📝 教训
- 下次怎么避免
```
