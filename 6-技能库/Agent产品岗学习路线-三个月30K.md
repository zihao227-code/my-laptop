# Agent 产品岗 30K 学习路线：三个月从零到 Offer

## 📌 一句话总结

三天上手是标题党，一般人没三个月沉淀根本行不通。**第三个月最关键 — 做 2-3 个完整 Agent 项目**，从用户调研到 PRD 到 MVP 走完整闭环。

---

## 📅 三个月路线图

### 第一个月：AI 前置知识

| 模块 | 内容 |
|------|------|
| 基础概念 | 大模型、LLM vs Agent 区别、工具调用、思维链 |
| Agent 组件 | 组件和原理、工具怎么调用、思维怎么设计 |
| 行业输入 | AI 产品案例分析、行业报告、技术博客 |

### 第二个月：Agent 核心能力进阶

| 模块 | 产品视角的关键问题 |
|------|------|
| Prompt 工程 | 怎么设计稳定的提示词母版？ |
| RAG 原理 | 解决什么问题？怎么召回？怎么切片排序？ |
| Agent 架构 | 有哪些经典设计范式和框架？ |

### 第三个月：产出完整 Agent 项目（最关键）

做 2-3 个实战，从易到难：

```
用户调研 → PRD → 原型 → Vibe Coding 做出 MVP
```

每个项目走完整闭环，不是只搭 Demo 录屏。
