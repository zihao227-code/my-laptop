# 💡 灵感库 — Inspiration

> 一闪而过的想法、待验证的假设、碎片化的思考

---

## 什么是"灵感"？

- **突然冒出的想法** — 洗澡时、散步时、睡前
- **待探索的方向** — "这个好像可以深入研究"
- **待解决的问题** — "为什么这个总是出错？"
- **想尝试的实验** — "试试把 A 和 B 组合起来会怎样"

---

## ⚡ 核心原则：快速捕获，延迟整理

```
捕获（10秒）  →  记录（30秒）  →  整理（每周）
    ↓                ↓                ↓
 抓到念头        一句话+标签      转化/归档/删除
```

---

## 📝 灵感笔记模板（极简）

```markdown
---
tags: [灵感]
created: YYYY-MM-DD
status: 待处理 | 已转化 | 已放弃
---

# 一句话标题

## 💡 原始想法
（写下当时想到的，不需要整理）

## 🤔 为什么觉得有意思

## 🔗 关联
- [[可能相关的笔记]]
```

---

## 🔄 定期清理流程

每周花 15 分钟过一遍灵感库：

| 处理方式 | 适用情况 |
|----------|----------|
| → 创建项目 | 想法成熟，可以落地 |
| → 补充到技能库 | 是一个技术点/方法论 |
| → 补充到资源库 | 可以找相关资料深入 |
| → 保留观察 | 暂时不行动但有点意思 |
| → 删除 | 回头看觉得没价值 |

---

> ⚡ **灵感是火花，不抓住就灭了 — 先记下来，好坏以后再判**
