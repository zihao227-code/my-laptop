# AI Coding 学习之旅

## 关于我
- **身份**: 软件公司 BA（Business Analyst）
- **编程基础**: 零基础/入门
- **学习目标**:
  1. 用 AI 工具提升 BA 工作效率
  2. 能独立用 AI 做原型/Demo
  3. 转行/进阶为 AI 产品经理（AI PM）
  4. 了解 AI 原理和应用边界
- **长期愿景**: 🚀 **一人公司** — 用 AI + Coding 能力，一个人完成从需求→设计→开发→上线的全链路

## 学习方向
- 🎯 AI 辅助编程实战（Claude Code, Cursor 等工具）
- ✍️ Prompt Engineering（提示词工程）
- 🧠 AI 基础概念（LLM, RAG, Agent, Token 等）
- 📊 用 AI 做数据分析（后续）

## 学习原则
1. **概念 + 实践并重**: 每学一个概念，马上动手练习
2. **从 BA 视角出发**: 以熟悉的业务场景作为练习素材
3. **渐进式难度**: 从简单 Prompt 到完整项目
4. **记录与复盘**: 每次学习后写笔记，记录关键收获和疑问

## 使用 Claude Code 的方式
- 本文件夹不创建大型项目，聚焦学习和实践
- 每个学习主题创建一个对应的笔记文件
- 实践代码放在 practice/ 下的对应目录
- 随时向 Claude 提问，把对话当作 1v1 导师
