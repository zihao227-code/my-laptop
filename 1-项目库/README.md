# 📁 项目库 — Projects

> 有明确目标、截止日期和交付物的短期任务

---

## 什么是"项目"？

- 有明确的**目标**和**产出物**
- 有（大致）的**截止时间**
- 需要多个步骤才能完成
- 完成后可以**归档**

**示例**：
- ✅ 「搭建个人博客」→ 有明确交付物
- ✅ 「准备下季度技术分享」→ 有截止日期
- ❌ 「学习前端」→ 这是领域/技能，放入技能库

---

status: 规划中 | 进行中 | 已完成 | 暂停
---

## 🔄 生命周期

```
规划中 → 进行中 → 已完成 → 归档到 [[4-归档库]]
              ↘ 暂停 → 定期审视是否重启
```
