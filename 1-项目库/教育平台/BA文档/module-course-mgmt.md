# 模块规格：课程管理（P0）

> 版本：v1.0 | 日期：2026-07-06 | 基于 Plan v1.0

---

## 使用者

- **Editor**（唯一有权限的角色，Trainer 和 Learner 不可见此模块）

---

## 功能清单

### 1. Listing 页面（默认入口）
- 课程以**表格**形式展示，字段：课程名称、类型、状态、老师、排期数量、价格、标签、创建时间
- **搜索**：按课程名称 + description 搜索（模糊搜索后期做）
- **筛选**：按状态 / 类型 / 老师 / 标签筛选
- **排序**：所有列均支持点击排序，默认按创建时间倒序
- **分页**：支持 10/20/50 条/页，用户可选
- **批量操作**：支持多选 → 批量归档 / 批量发布
- **单行操作**：Publish / Edit / Archive（软删除）→ Archive 后可恢复

### 2. 创建课程（独立页面）
- 基础信息：名称、描述、封面图（支持上传，未上传用默认图）、类型(online/offline/hybrid)、标签、原价
- 排期配置：增删改查上课时间（分阶段规则见下）
- 关联老师：从预设老师列表中选择（老师一旦被课程关联，不可删除）
- 保存选项：
  - **Save as Draft**：保存草稿，不校验必填项
  - **Cancel**：放弃，返回 Listing
  - **Save & Publish**：发布前校验所有必填项 → 通过才发布

### 3. 编辑课程
- 进入 Edit 页面 = 创建草稿（标记为"变更型草稿"）
- 草稿未发布前：原有课程内容仍对外生效
- 发布草稿：校验必填项 → 通过后覆盖原课程内容
- 版本记录：记录第几版本，支持查看历史版本（暂不做回滚）

### 4. 草稿管理
- 草稿列表：标注类型（"新建" / "变更 C001"）
- 可从草稿列表继续编辑或删除草稿

### 5. 删除 / 归档
- 课程本身：软删除（归档）→ 不可见但数据保留 → 支持恢复
- 排期删除（分阶段）：
  - 阶段一：未有人报名 → 直接删除，不留痕
  - 阶段二：已报名未上课 → 软删 + 自动退费 + 更新订单状态
  - 阶段三：已报名已上课 → 不允许删除

### 6. 状态流转
- `draft → published → in_progress → completed`
- 允许在 Listing 页面操作（Publish 按钮）
- 允许在 Edit 页面操作（Save & Publish）

### 7. 必填项校验
- 发布/Create 前自动校验，未通过则高亮缺失字段 + 提示信息
- 已发布课程：所有必填项必须完整，否则无法提交

### 8. 权限硬约束
- 仅 Editor 可访问课程管理模块
- Trainer 和 Learner 无入口、无链接、URL 直接访问也被拒绝

### 9. 老师关联约束
- 老师 → 课程：一旦关联，老师不可被删除
- 必须"先替换再解除"：解除关联 = 指定新老师替换旧老师
- 不允许"从有配置变为无配置"
