# 🏗️ 资产库 — Assets

> 长期积累的可复用资产：模板、脚本、配置、方法论

---

## 什么是"资产"？

- 可以**跨项目复用**的东西
- 需要**持续迭代维护**
- 具有**长期价值**，不会过时

**示例**：
- ✅ 代码片段模板、项目启动脚本
- ✅ 个人配置（shell、编辑器、快捷操作）
- ✅ 写作模板、会议纪要模板
- ✅ 个人原则、决策框架

---

## 📂 子目录

```
2-资产库/
├── 模板/         # 各类文档模板
├── 脚本/         # 常用自动化脚本
├── 配置/         # 开发环境、工具配置
├── 方法论/       # 个人工作流、决策框架
└── 组件/         # 可复用的代码/设计组件
```

---

## 💡 沉淀原则

> 第二次做同样的事 → 提炼成资产

当你发现自己重复做某件事时，就是沉淀资产的最佳时机：
1. 第一次：边做边记录
2. 第二次：参考上次记录，补充完善
3. 第三次：提炼成模板/脚本，放入资产库

---

> 💡 **资产库是"可生长的"** — 不是一次性创建完，而是在实践中逐渐生长出来
