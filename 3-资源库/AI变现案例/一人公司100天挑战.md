# AI 一人公司 100 天挑战

## 📋 人物画像

| 维度 | 描述 |
|------|------|
| 起点 | 没背景、没人脉、普通大学、普通工作 |
| 原收入 | 月入一万多 |
| 辞职动机 | 算了一笔账：普通路径要工作40年才能还完房贷，父母在变老 |
| 安全垫 | 攒了一年的生活费才辞职 |
| 方向 | 瞄准"AI + 具体行业" |

---

## 🎯 变现模式

**类型**：一人公司（AI + 本地商家服务）

**具体做法**：
- 线上：AI 生成内容发到自媒体平台，测试市场兴趣
- 线下：地推，用 AI 工具帮商家做具体的事（生图、内容等）
- 最终跑通：线下地推拿到几家店铺合作意向，成为核心主项目

---

## 💰 启动成本

| 类型 | 估算 |
|------|------|
| 时间 | 从零到跑通约 4 个月（含 2 个月弯路） |
| 金钱 | 一年生活费 + 两个不必要 AI 课程的费用 |
| 技能 | 生图、生视频（宠物行业方向），不是全栈 AI |

---

## 🔑 关键决策

| 决策 | 为什么重要 |
|------|------|
| **辞之前攒够一年生活费** | 确保失败后还能继续，不会因为经济压力放弃 |
| **线下地推而非只做线上** | 线上没反馈，线下反而拿到合作意向 — 真正的需求在真实场景里 |
| **开启"100天挑战"公开承诺** | 倒逼自己每天出成果，同时解决心态问题 |
| **低成本试错** | 失败了不影响生活，可以反复尝试 |

---

## 🕳️ 踩过的坑（最有价值的部分）

### 坑一：用"学习"逃避面对市场

```
需要学的：生图 + 生视频 （仅2项，宠物方向）
实际学的：工作流 + 大模型 + 配音 + 数字人 + 报了两个课程
          + 收藏了几百个工具链接（大半已过时）
结果：耗时两个月，离目标越来越远
```

**根因**：以为"先学会 AI，才能用 AI 做事"。但这句话反了 — 应该是"先知道要做什么事，再去学需要的那部分 AI"。

**关联**：与 [[AI赚钱思维-以终为始]] 的核心观点完全一致 — 追着 AI 学 AI，是一条没有终点的路。

### 坑二：闷头深造，不测试市场

用了两个月学会 AI 生成东西 → 做产品 → 推向市场 → 没人要。

**根因**：没验证"市场需不需要"就开始做。学会 AI ≠ 会用 AI 跑完整项目。

**教训**：先测试市场 → 有人感兴趣 → 再做 MVP → 达成成交 → 再完善。

### 坑三（不算坑，算教训）：线上没反馈别死磕

线上自媒体发了很多内容（干货型/痛点型/分享型 三种都有），都没明显反馈。但**线下地推**反而很快拿到合作意向。真实需求不一定在线上，有时候就在街头巷尾。

## ✅ 可复用方法

### 项目筛选

从 100+ 个 AI 项目里挑了 2 个适合自己的：
- 标准：技能匹配 + 低成本可启动 + 有明确付费方

### 市场测试三步

| 步骤 | 做法 |
|------|------|
| 1. 生成素材 | 用 AI 生成图片、样式或初步 Demo |
| 2. 发布测试 | 发到自媒体平台，三种内容类型：干货型、痛点型、分享型 |
| 3. 看反馈 | 有没有人评论、点赞、私信？没有就换方向 |

### 心态倒逼机制

```
公开承诺（100天挑战，每天发视频）
        ↓
必须每天有内容可发
        ↓
必须每天推进项目
        ↓
正循环：忙起来 → 不焦虑 → 拿反馈 → 继续忙
```

### 避坑清单

1. **学会 AI ≠ 会用 AI 做事**。只学目标需要的那部分
2. **低成本试错**。每次失败不影响生活，可以反复试
3. **先测市场，再做产品**。有人感兴趣才投入时间完善
4. **不要三天打鱼两天晒网**。持续发、持续做才有用
5. **不要没方向就辞职**。先用业余时间摸索
6. **AI 知识学不完**。下个月又有新名词，只学对你有用的
7. **执行力是前提**。不要等学完再做，边做边学
