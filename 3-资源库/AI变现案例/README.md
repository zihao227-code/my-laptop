| 维度 | 说明 |
|------|------|
| **人物画像** | 背景、起点、资源 |
| **变现模式** | 一人公司 / 外包接单 / SaaS / 自媒体 / 培训 / 其他 |
| **启动成本** | 时间、金钱、技能门槛 |
| **关键决策** | 哪一步做对了，为什么 |
| **踩过的坑** | 反面教材往往更有价值 |
| **可复用方法** | 能直接套用的步骤和原则 |
| **与你的关联** | 对你当前阶段的参考意义 |
