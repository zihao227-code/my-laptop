# 📚 资源库 — Resources

> 外部资料整理：文章、教程、工具、书摘、视频笔记

---

## 什么是"资源"？

- 来自**外部**的信息和知识
- 经过**筛选和消化**，不是简单收藏
- 按**主题分类**，方便检索

**示例**：
- ✅ 读完一篇文章后的核心要点笔记
- ✅ 一个好用工具的用途和使用技巧
- ✅ 书籍的读书笔记和行动清单
- ❌ 随手保存的链接（先放灵感库）
- ❌ 没读过没消化的原文收藏（这是书签，不是知识）

---

## 📂 分类建议

```
3-资源库/
├── 技术文章/     # 技术博客、论文、RFC
├── 书籍笔记/     # 读书摘要、心得
├── 视频课程/     # 视频教程笔记
├── 工具推荐/     # 好用的工具及其最佳实践
├── 设计参考/     # 优秀设计案例、UI/UX 参考
└── 行业洞察/     # 行业报告、趋势分析
```

rating: 1-5

## 📌 一句话总结
