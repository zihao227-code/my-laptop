# AI Coding 学习路线图

## Phase 0: 环境准备（当前阶段）
- [ ] 了解 Claude Code 的基本用法
- [ ] 熟悉常用命令（/help, /clear 等）
- [ ] 了解文件结构和学习流程

## Phase 1: AI 基础概念 🧠
预计时间：1-2 周

### 1.1 什么是 LLM（大语言模型）
- [ ] 理解 LLM 的基本原理（预测下一个 token）
- [ ] 了解主流模型：Claude, GPT, Gemini 等
- [ ] 关键概念：Token, Context Window, Temperature

### 1.2 AI 能做什么 & 不能做什么
- [ ] AI 的优势领域和典型应用
- [ ] AI 的局限性（幻觉、上下文限制、知识截止日期）
- [ ] 从 BA 视角：AI 如何改变需求分析、文档编写、原型制作

### 1.3 进阶概念概览
- [ ] RAG（检索增强生成）
- [ ] Agent（智能体）
- [ ] MCP（模型上下文协议）
- [ ] Fine-tuning vs. Prompt Engineering

## Phase 2: Prompt Engineering ✍️
预计时间：2-3 周

### 2.1 Prompt 基础
- [ ] 什么是 Prompt Engineering
- [ ] 好 Prompt 的要素（角色、上下文、格式、约束）
- [ ] 常见技巧：Few-shot, Chain-of-Thought, 结构化输出

### 2.2 Prompt 模式与实践
- [ ] 角色扮演模式（"你是一个资深 BA..."）
- [ ] 分步推理（"让我们一步一步思考"）
- [ ] 格式化输出（JSON, Markdown, 表格）
- [ ] 迭代优化 Prompt 的方法

### 2.3 BA 场景 Prompt 实战
- [ ] 用 AI 编写和优化用户故事
- [ ] 用 AI 生成验收标准（Acceptance Criteria）
- [ ] 用 AI 分析需求文档
- [ ] 用 AI 生成流程图/时序图描述

## Phase 3: AI 辅助编程实战 💻
预计时间：4-6 周

### 3.1 编程基础速览
- [ ] 理解前端/后端的基本概念
- [ ] HTML/CSS/JavaScript 入门
- [ ] 理解什么是 API、数据库

### 3.2 AI 编程工具
- [ ] Claude Code 深度使用
- [ ] 用 AI 阅读和理解代码
- [ ] 用 AI 写代码、修 Bug
- [ ] 用 AI 解释技术概念

### 3.3 从零搭建原型
- [ ] 项目1：用 AI 搭建一个简单的需求管理页面
- [ ] 项目2：用 AI 搭建一个数据看板
- [ ] 项目3：用 AI 搭建一个自动化工作流

### 3.4 进阶实践
- [ ] 用 AI 写自动化脚本
- [ ] 用 AI 处理 Excel/CSV 数据
- [ ] 用 AI 生成 API 文档和接口定义

## Phase 4: 综合应用 🚀
预计时间：持续进行

- [ ] 用 AI 完成 BA 日常工作全流程
- [ ] 独立完成一个端到端的原型项目
- [ ] 建立自己的 Prompt 模板库
- [ ] 持续跟进 AI 领域最新发展

---

## 学习节奏建议
- **每天**：30-60 分钟学习 + 练习
- **每周**：完成 2-3 个小主题
- **每个 Phase 结束**：做一个综合练习，写一篇学习总结
