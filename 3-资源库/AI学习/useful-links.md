## 概念速查

| 术语 | 一句话解释 |
|------|-----------|
| LLM | 大语言模型，读了海量文本的"文本预测器" |
| Token | AI 的最小文字单位，≈0.5个中文字 |
| Context Window | AI 一次能"看到"的最大内容量 |
| Prompt | 发给 AI 的指令/问题 |
| RAG | 让 AI 先查资料再回答，减少幻觉 |
| Agent | 能使用工具的自主 AI |
| MCP | AI 连接外部工具的协议标准 |
| Hallucination | AI 编造假信息的现象 |
| Fine-tuning | 用特定数据进一步训练 AI |
| Embedding | 把文本转成数字向量，用于语义搜索 |

> 💡 **建议**：遇到不理解的术语时，随时在对话中问 Claude，这是最快的学习方式。
